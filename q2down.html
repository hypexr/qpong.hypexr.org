Q2Pong Downloads

 

Here you will find the basic files you will need to get QPong set up or to run a server.  I also recommend that you download the additional maps and skins.  :)

To install qpong:

1. Download the QPong client pack (1.0) completely, and put it in a temporary directory of your choice.

2. Use WinZip to extract the files. Make sure you have "Keep directories" checked off when unzipping. Unzip it to your ROOT directory when you have this checked (example: if quake2 is at c:\quake2, unzip it to c:).

3. Find a QPong server using GameSpy, PingTool, or click on the server button and kick some QPong butt! :)

To install maps:

1. Download the map packs.

2. Extract them into your \quake2\qpong\maps\ directory.
